# Fluir

A mobile app prototype for recently diagnosed mothers with ADHD.

---

## About

Fluir combines personalised task management, a psychoeducation 
library, and daily emotional check-ins in one platform designed 
around how the ADHD brain actually works.

This is a minimum viable prototype built for user testing via 
Lyssna as part of Assessment 3 for 2043IBA Innovation, 
Creativity and Entrepreneurship at Griffith University, 
Trimester 1 2026.

## The name

Fluir comes from the Spanish and Portuguese verb meaning 
"to flow" — from the Latin fluere, the same root as fluid, 
fluent, and flow.

It is not a noun. It is a verb. Not a state you arrive at 
but something you do continuously. The ADHD brain does not 
need to be fixed. It needs to find its flow.

Every design decision in Fluir — the colour, the typography, 
the shape language, the tone, the features — flows from 
this single idea. Work with the brain, not against it. 
Move naturally, without resistance.

Finally, something that gets you.
